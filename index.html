Install in 15 minutes!

1. Download and put the continue folder on your website!

2. Get a Stripe account and put the relevant keys in the ajax.php file (I recommend using the testing keys first and making sure it works for you).

3. Use this HTML:

<script src="continue/script.js"></script>
<script>new Continue;</script>

To customize your Continue, look at the code below!
